# Il Decisore

Diciamoci la verità: scegliere cosa mangiare è spesso la parte più difficile della serata.
Ho scritto questo piccolo script in Python proprio per risolvere questo problema (e per esercitarmi un po' con il codice).

### Come funziona
È molto semplice:
- Se hai già delle idee ma non sai decidere, gliele scrivi e lui sceglie per te.
- Se hai il vuoto totale, ti propone lui dei grandi classici.

### Dietro le quinte
È il mio **primo progetto caricato su GitHub**!
Mi sono divertito a creare l'effetto "suspense" con i puntini di caricamento (usando `time.sleep`) per non dare la risposta troppo in fretta.

Il codice è ancora semplice, ma fa il suo dovere quando l'indecisione colpisce. ;)
